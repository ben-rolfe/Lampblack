4. Profiles

Each Propagate Audio source is linked to a profile. A profile is an asset file that contains various settings for Propagate. The default profile is there to get you up and running, but for more control you can create new ones.

To create a new profile simply duplicate the default and rename it.

Settings

A profile has settings split into three sections, Volume, Frequency and Occlusion.

Volume Settings

The volume settings control how the volume of audio sources change with respect to distance.

Responsiveness

The responsiveness indicates how quickly the volume changes as the distance to the listener changes. Lower values indicate a smoother/slower transition. Higher values indicate a sharper/faster transition.

Min Volume

The minimum volume a Propagate Audio Source can reach.

Max Volume

The maximum volume a Propagate Audio Source can reach.

Frequency Settings

The frequency settings control how the frequency range is shortened with respect to curvature of the propagation. The more corners the sound has to propagate past, the lower the frequency range.

Responsiveness

The responsiveness indicates how quickly the frequency cut-off changes as the curvature changes. Lower values indicate a smoother/slower transition. Higher values indicate a sharper/faster transition.

Min Cut-off

The minimum frequency cut-off. This number indicates the minimum frequency that will be heard. For instance a value of 100 means only frequencies 100hz and lower will be heard.

Max Cut-off

The maximum frequency cut-off. This number indicates the maximum frequency that will be heard. For instance a value of 10,000 means only frequencies 10,000hz and lower will be heard.

Max Angle

The maximum change in angle the sound can propagate before it reaches the minimum cut-off amount.

Curve

The falloff curve for frequency response. The x-axis corresponds to the curvature of the path, where 0 is no change in angle (direct sound) and 1 is the maximum angle. The y-axis corresponds to the frequency cut-off, where 0 is the minimum cut-off value and 1 is the maximum cut-off value.

Occlusion Settings

The occlusion settings control how the volume and frequency range is diminished as it travels through solid objects.

Volume Reduction

This value indicates how much of the volume is reduces as it passes through a solid object. A value of 1 indicates a 100% volume reduction (no sound gets through). A value of 0 indicates 0% volume reduction (all sound gets through).

Frequency Reduction

This value indicates how much of the higher frequencies travel through solid objects. A value of 1 indicates 100% percent of the frequencies pass through (no change in frequency range). A value of 0 means 0% of frequencies pass through (no sound gets through). A value of 0.5 means 50% of the frequency range gets through, if the frequency range was 0-10000hz before it passed the solid object then it would be 0-5000hz afterwards.

Miscellaneous settings

Perceived Position Responsiveness

Indicates how quickly a Propagate Audio Source moves to the perceived position. A lower value indicates a smoother/slower transition. A higher value indicates a sharper/faster transition.
Setting this value too high may cause the pitch of sounds to bend due to the doppler effect.

Perceived Position Effect

Indicates how much the perceived position of an audio source comes into effect. Unity will pan a sound left or right depending on the angle to the listener. However the geometry of an environment can cause that sound to be perceived as coming from a different direction.

0 = 0% (no percieved position effect).
1 = 100% (full percieved position effect).
